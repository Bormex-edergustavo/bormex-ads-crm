Politica de privacidad

Ultima actualizacion: 29 de mayo de 2026

Bormex Ads CRM es una herramienta interna para medir el rendimiento de anuncios, conversaciones y ventas. Esta politica explica que datos se procesan y para que se usan.

Datos que podemos procesar

  • Datos de anuncios de Meta, como campanas, conjuntos, anuncios, gasto, presupuesto e identificadores.
  • Datos de conversaciones recibidas por canales conectados, como numero de telefono, nombre visible, fecha, canal y contenido del mensaje.
  • Ventas registradas manualmente, como numero de WhatsApp, productos vendidos, monto y fecha.
  • Datos tecnicos necesarios para operar el panel, como estado de sincronizacion y registros de eventos.

Uso de la informacion

La informacion se usa para atribuir ventas a anuncios, calcular costo por mensaje, costo por venta, ROAS y recomendaciones operativas sobre presupuesto o rendimiento. No se vende informacion personal.

Servicios utilizados

El sistema puede usar servicios de Meta, WhatsApp Business Platform, Supabase y GitHub Pages para autenticar, recibir eventos, guardar datos y mostrar el panel.

Conservacion y eliminacion

Los datos se conservan mientras sean necesarios para operar el CRM y analizar ventas. Si un cliente solicita eliminar o corregir sus datos, Bormex puede atender la solicitud desde sus canales oficiales de contacto.

Seguridad

El panel usa acceso protegido y guarda secretos de integracion fuera del repositorio publico. El acceso operativo debe limitarse a personal autorizado.
